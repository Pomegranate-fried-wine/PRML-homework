## 引言

在数据科学与工程建模中，选择合适的数学模型架构比单纯提高算法迭代次数更为重要。本实验经历了从单一线性拟合到非线性三角函数拟合，再到“线性 + 正弦”复合模型拟合的探索过程。

实验发现，忽视数据的全局趋势会导致非线性模型陷入局部最优。通过引入**线性补偿项**并结合 **FFT (快速傅里叶变换)** 频谱分析进行参数预初始化，成功实现了对复杂信号的高精度重构。本实验构建了复合模型，探讨了从杂乱样本中分离线性特征与非线性特征的方法。实验结果表明，该模型在均方误差（MSE）表现上显著优于单一维度的拟合方案。

**实验核心目标：**
1.  对比不同优化算法在同一线性模型下的收敛性与泛化能力。
2.  分析数据的内在结构特征，判断线性模型的适用性。
3.  构建更高表达能力的非线性模型，并进行实验验证。

## 一、 三种线性拟合模型

本阶段目标是使用三种经典优化算法对给定数据进行线性拟合（模型：$y = wx + b$），并对比其在收敛效率与误差表现上的差异。

**目标函数（MSE）：**
$$J(w, b) = \frac{1}{n} \sum_{i=1}^{n} (y_i - (wx_i + b))^2$$

### 1.1 拟合算法原理简述

* **最小二乘法 (OLS):** 通过解析解直接计算精确参数。通过正规方程 $(X^T X)^{-1} X^T y$ 一步到位求得全局最优解。计算稳定但大规模矩阵求逆开销大。
* **梯度下降法 (GD):** 沿着 MSE 梯度的反方向逐步迭代更新。通用性强，但对学习率 $\eta$ 敏感，易陷入局部最优。
* **牛顿法 (Newton's Method):** 利用二阶导数（Hessian 矩阵）信息，通过泰勒展开的二次项近似目标函数，收敛速度呈平方级，但计算开销较高。

### 1.2 拟合结果对比

| 算法类型 | 训练误差 (MSE) | 测试误差 (MSE) |
| :--- | :--- | :--- |
| **最小二乘法** | 0.613402 | 0.595043 |
| **牛顿法** | 0.613402 | 0.595043 |
| **梯度下降法** | 0.618825 | 0.594021 |

**线性模型的局限性分析：**
观察散点图发现，数据点并非均匀分布在直线两侧，而是呈现出明显的**周期性波动**。虽然线性模型捕捉到了整体上升趋势，但 MSE 较高（约 0.6），说明线性模型漏掉了数据中最重要的结构化特征。实验验证了 OLS 与牛顿法结果一致，说明优化方法不是瓶颈，**模型假设**才是关键。

## 二、 复合模型构建

### 2.1 复合模型设计

基于对残差呈现结构性分布的观察，提出**线性 + 三角复合拟合函数模型**：
$$y = kx + a \cdot \sin(bx + c) + d$$

* **线性项 ($kx$):** 解释数据随 $x$ 轴整体抬升的趋势。
* **正弦项 ($a \cdot \sin(bx + c)$):** 解释数据在趋势线上方的周期性波动。

### 2.2 实验结果与分析

引入复合模型并配合 **FFT** 进行频率初始化后，拟合质量发生质变。训练集 MSE 降至 **0.4343**，测试集 MSE 为 **0.4447**。
