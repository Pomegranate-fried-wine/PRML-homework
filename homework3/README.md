# 多变量时间序列PM2.5预测实验报告（具体实验报告请看PDF文件）

---

# 1. 实验背景与问题定义

空气污染预测是环境科学与智能城市研究中的重要问题，其中细颗粒物 PM2.5 的预测具有重要现实意义。本实验基于北京空气质量监测数据，构建多变量时间序列预测模型，研究气象因素与污染物浓度之间的时序依赖关系。

本研究将问题定义为：

> 在给定过去12小时多变量气象与污染观测的情况下，预测下一时刻 PM2.5 浓度。

该问题属于典型的多变量时间序列回归问题。

---

# 2. 数据集与预处理

## 2.1 数据来源

数据集来源于 Kaggle：

* 文件：LSTM-Multivariate_pollution.csv
* 时间跨度：多年逐小时观测数据
* 特征变量：

| 类型   | 变量       |
| ---- | -------- |
| 污染物  | PM2.5    |
| 气象变量 | 温度、气压、露点 |
| 风相关  | 风向、风速    |
| 降水变量 | 降雪、降雨    |

---

## 2.2 数据预处理

为了适配模型输入，进行了以下处理：

### （1）时间索引处理

将 date 字段转换为时间索引，用于保持时间顺序。

### （2）类别变量编码

风向变量 wnd_dir 使用 LabelEncoder 进行数值化处理。

### （3）缺失值处理

采用 0 填充缺失数据。

### （4）数据归一化

使用 MinMaxScaler 将所有特征缩放至 [0,1] 区间，避免量纲影响模型训练。

---

# 3. 时间序列构造

构建监督学习样本：

Time Series Forecasting

采用滑动窗口方法：

* 输入：过去24小时（look_back = 24）的多变量特征
* 输出：当前时刻 PM2.5

形式定义如下：

* 输入：
  [
  X_t = [x_{t-24}, ..., x_{t-1}]
  ]

* 输出：
  [
  y_t = PM2.5_t
  ]

---

# 4. 数据集划分策略

采用**时间顺序划分**：

* 前 80%：训练集
* 后 20%：测试集

该方式避免数据泄露，符合时间序列建模规范。

---

# 5. 模型结构设计

本实验采用基于循环神经网络的深度学习模型：

Long Short-Term Memory

## 5.1 模型结构

模型采用**堆叠LSTM结构**：

```
Input (24×8)
→ LSTM(64, return_sequences=True)
→ Dropout(0.2)
→ LSTM(32)
→ Dropout(0.2)
→ Dense(1)
```

---

## 5.2 结构说明

* 第一层 LSTM（64单元）：

  * 提取短期时间依赖特征
  * 输出完整序列（return_sequences=True）

* 第二层 LSTM（32单元）：

  * 进一步提取高级时间抽象特征
  * 压缩时序信息

* Dropout层：
  Dropout

  * 防止过拟合
  * 提高泛化能力（drop rate = 0.2）

* Dense层：

  * 输出最终 PM2.5 预测值

---

# 6. 训练策略

## 6.1 损失函数与优化器

* 损失函数：MAE（Mean Absolute Error）
* 优化器：Adam

---

## 6.2 Early Stopping机制

引入早停策略：

Early Stopping

参数设置：

* patience = 5
* monitor = val_loss
* restore_best_weights = True

作用：

* 防止过拟合
* 自动选择最优模型

---

## 6.3 训练设置

* epochs = 20
* batch_size = 64
* shuffle = False（保持时间顺序）

---

# 7. 实验结果与分析

## 7.1 Loss收敛情况

训练过程中：

* train loss 持续下降
* validation loss 在中期趋于稳定
* EarlyStopping 在最优点提前终止训练

说明模型已收敛且避免过拟合。

---

## 7.2 预测性能评估

采用 RMSE 作为评价指标：

[
RMSE = \sqrt{\frac{1}{n}\sum (y - \hat{y})^2}
]

实验结果表明：

* 模型能够较好拟合 PM2.5 时间变化趋势
* 在高波动区域存在一定误差

---

## 7.3 预测结果可视化分析

### （1）整体预测结果

模型能够较好捕捉长期趋势变化，但在极值点存在一定偏差。

---

### （2）局部放大分析

在前200个样本中：

* 预测曲线与真实曲线趋势一致
* 存在轻微滞后现象（LSTM典型特性）

---

### （3）误差分布

误差呈近似正态分布：

* 大部分误差集中在0附近
* 少量极端误差来自污染突变情况

---

# 8. 讨论

## 8.1 模型优势

* 能够建模多变量时间依赖关系
* 堆叠LSTM增强特征表达能力
* Dropout提升泛化能力
* EarlyStopping提高训练稳定性

---

## 8.2 局限性

* 对突发污染变化响应较慢
* 未引入外部环境变量（如工业排放）
* 单步预测模型，缺乏长期预测能力

---

# 9. 结论

本实验构建了基于堆叠LSTM的多变量时间序列预测模型，用于PM2.5浓度预测。实验结果表明，该模型能够有效学习气象与污染之间的时序关系，在时间序列预测任务中具有良好表现。
