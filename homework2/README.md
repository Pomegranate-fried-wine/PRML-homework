# Homework2：3D Moons 数据集的非线性分类研究（具体详见PDF文件）

**实验主题**：对比决策树、集成学习（AdaBoost）与支持向量机（SVM）在 3D 空间非线性流形数据上的分类性能。

---

## 1. 实验目的
1. 探究 **Decision Tree** 在 3D 空间中的切分逻辑及其对非线性边界的拟合能力。
2. 验证集成学习算法 **AdaBoost** 如何通过组合弱分类器提升模型泛化性能。
3. 对比 **SVM** 在不同核函数（Linear, Poly, RBF）下的空间映射能力与分类精度。
4. 通过 3D 决策曲面可视化，深入理解各算法的几何分类本质。

## 2. 数据集描述
* **数据形态**：生成的 3D Make Moons 数据集，呈现两个相互交织的半月形（螺旋状）分布。
* **特征维度**：3维 ($X, Y, Z$)。
* **样本规模**：训练集 2000 个样本，测试集 1000 个样本（C0 与 C1 各占 50%）。
* **噪声水平**：设置 `noise=0.2`，模拟现实中数据重叠和干扰的情况。

## 3. 实验方法与模型设置

### 3.0 实验假设
* **设置**：为了避免过拟合，同时直观比较各算法的效果，本实验对迭代轮次和深度进行归一化。

### 3.1 决策树 (Decision Tree)
* **设置**：限制 `max_depth=5`。
* **逻辑**：采用基于信息增益（Gini系数）的轴平行切分。

### 3.2 AdaBoost + DecisionTrees
* **设置**：以 `max_depth=5` 的决策树为基学习器，迭代次数 `n_estimators=100`，学习率 `0.1`。
* **逻辑**：通过对分类错误样本增加权重，逐步修正边界误差。

### 3.3 支持向量机 (SVM)
实验选取了三种具有代表性的核函数：
1.  **Linear Kernel**：尝试在 3D 空间寻找最优超平面。
2.  **Polynomial Kernel (degree=3)**：通过三阶多项式特征映射进行非线性拟合。
3.  **RBF Kernel (Radial Basis Function)**：利用高斯径向基函数，将数据映射到高维超平面，处理复杂的局部流形分布。





## 4. 实验结果对比

实验测试集共 1000 个样本（C0: 500个，C1: 500个）。根据程序实际运行输出，各模型的表现如下表所示：

### 4.1 综合性能统计表

| 算法模型 | 准确率 (Accuracy) | 混淆矩阵 $[TP, FP], [FN, TN]$ | 结果分析 |
| :--- | :---: | :---: | :--- |
| **SVM (Linear)** | **0.6770** | `[[335, 165], [158, 342]]` | **表现最差**。线性超平面无法切分交织的 3D 数据。 |
| **SVM (Polynomial)** | **0.8600** | `[[480, 20], [120, 380]]` | 具有一定曲面拟合力，但对 C1 类的召回率（Recall）较低。 |

---

## 5. 讨论与分析：为什么某些算法表现更好？

### 5.1 线性模型的本质局限 (SVM Linear: 0.6770)
从 0.6770 的准确率可以看出，线性核几乎处于“半失效”状态。在 3D Moons 数据中，两个类别的点在空间中是相互扭曲、嵌套的。线性 SVM 只能在空间中划出一个**平整的切面**。无论这个平面如何摆放，都不可避免地会将大量属于 C0 的点划入 C1 区域，反之亦然。这证明了线性模型无法处理高维非线性特征。

### 5.2 决策树的“阶梯效应”与 AdaBoost 的进化
* **单棵决策树 (0.8910)**：混淆矩阵中 `[41, 68]` 的误判主要发生在两个月牙交界处。由于决策树只能进行垂直于坐标轴的切分，它在尝试拟合月牙的弧形边缘时，只能用一个个立方体去填充，导致边界不够平滑。
* **AdaBoost (0.9610)**：通过将 100 棵树进行加权集成，AdaBoost 相当于在 3D 空间中利用数千个微小的决策平面构建了一个**极其复杂的包围网**。它将总错误数从 109 个降低到了 39 个，极大地提升了对边缘噪声的鲁棒性。



### 5.3 SVM RBF 核的“降维打击” (0.9760)
RBF（径向基函数）核表现最好，其混淆矩阵显示仅有 24 个样本误判。
* **空间映射**：RBF 实际上是将 3D 数据映射到了**无限维特征空间**，在那个高维空间中，原本扭曲的数据变得线性可分。
* **几何形态**：在原始 3D 空间中，RBF 的决策边界表现为平滑的、像**胶囊外壳**一样的曲面，完美地包裹住了每一个月牙流形。相比于决策树的锯齿边界，这种平滑性使其在处理 `noise=0.2` 的模糊地带时更加精准。



---

## 6. 实验结论

1.  **非线性数据首选 RBF SVM**：实验证明，对于 3D Moons 这种具有明显几何流形特征的数据，基于距离相似度的 RBF 核函数具有天然的适配性。
2.  **集成学习（AdaBoost）极具价值**：即使基分类器（决策树）能力有限，通过集成学习也能在不改变模型性质的前提下，通过“群体决策”逼近最优性能。
3.  **模型复杂度需与数据匹配**：线性模型在此类问题上表现极差，说明在实际应用中，识别数据的空间分布特征并选择合适的核函数或集成策略是分类任务成功的关键。
