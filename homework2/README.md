# Homework2：3D Moons 数据集的非线性分类研究（具体详见PDF文件）

**实验主题**：对比决策树、集成学习（AdaBoost）与支持向量机（SVM）在 3D 空间非线性流形数据上的分类性能。

---

## 1. 实验目的
1. 探究 **Decision Tree** 在 3D 空间中的切分逻辑及其对非线性边界的拟合能力。
2. 验证集成学习算法 **AdaBoost** 如何通过组合弱分类器提升模型泛化性能。
3. 对比 **SVM** 在不同核函数（Linear, Poly, RBF）下的空间映射能力与分类精度。
4. 通过 3D 决策曲面可视化，深入理解各算法的几何分类本质。

## 2. 数据集描述
* **数据形态**：生成的 3D Make Moons 数据集，呈现两个相互交织的半月形（螺旋状）分布。
* **特征维度**：3维 ($X, Y, Z$)。
* **样本规模**：训练集 1000 个样本，测试集 500 个样本（C0 与 C1 各占 50%）。
* **噪声水平**：设置 `noise=0.2`，模拟现实中数据重叠和干扰的情况。

## 3. 实验方法与模型设置

### 3.0 实验假设
* **设置**：为了避免过拟合，同时直观比较各算法的效果，本实验对迭代轮次和深度进行归一化。

### 3.1 决策树 (Decision Tree)
* **设置**：限制 `max_depth=5`。
* **逻辑**：采用基于信息增益（Gini系数）的轴平行切分。

### 3.2 AdaBoost + DecisionTrees
* **设置**：以 `max_depth=5` 的决策树为基学习器，迭代次数 `n_estimators=100`，学习率 `0.1`。
* **逻辑**：通过对分类错误样本增加权重，逐步修正边界误差。

### 3.3 支持向量机 (SVM)
实验选取了三种具有代表性的核函数：
1.  **Linear Kernel**：尝试在 3D 空间寻找最优超平面。
2.  **Polynomial Kernel (degree=3)**：通过三阶多项式特征映射进行非线性拟合。
3.  **RBF Kernel (Radial Basis Function)**：利用高斯径向基函数，将数据映射到高维超平面，处理复杂的局部流形分布。

---

## 4. 实验结果对比

| 算法模型 | 测试集准确率 (Accuracy) | 边界特征描述 |
| :--- | :--- | :--- |
| **Decision Tree (Depth 5)** | ~89.1% | 阶梯状、块状边界 |
| **AdaBoost + DT** | ~96.1% | 边界更细腻，有效平滑了决策区域 |
| **SVM (Linear)** | ~67.7% | 单一的平整切面，无法处理弯曲数据 |
| **SVM (Polynomial)** | ~86.0% | 弯曲的超曲面，但在极端交织处精度受限 |
| **SVM (RBF)** | ~97.6% | 平滑的管道状/包裹状边界，最契合数据分布 |



---

## 5. 讨论与分析：为什么某些算法表现更好？

### 5.1 决策树的几何局限性
决策树本质上是利用**轴平行（Axis-aligned）**的不等式对空间进行切分。对于 3D Moons 这种圆润弯曲的流形数据，决策树必须使用大量的“直角弯”去逼近曲线，这导致了边界上的“阶梯效应”。在 `max_depth` 受限时，无法完美包裹月牙边缘。

### 5.2 AdaBoost 的提升原理
AdaBoost 通过集成 100 棵弱分类器，实现了从“粗糙积木”到“精细拼图”的转变。每一棵后续生成的树都针对前序树分错的边缘点进行专项优化。在 3D 决策曲面中可以看到，AdaBoost 生成的边界比单棵树更紧凑地贴合了月牙轮廓，显著提升了泛化精度。

### 5.3 SVM 核函数的降维打击
* **Linear 核的失效**：由于数据集在原始 3D 空间是**线性不可分**的，单纯靠一个平面（$Ax+By+Cz+D=0$）显然无法将两个交织的半月分开。
* **RBF 核的卓越性**：RBF 核通过衡量样本间的距离，实质上将数据映射到了无穷维空间。在 3D 可视化中，RBF 生成的决策曲面如同一个“胶囊”平滑地包裹住数据。这种**非线性、连续且平滑**的边界最符合 3D Moons 的物理分布规律，因此其准确率最高。



## 6. 实验结论
对于具有非线性流形特征的 3D 数据集，**SVM (RBF)** 凭借其自然的平滑边界表现出最强的分类性能。**AdaBoost** 紧随其后，证明了集成学习在修正弱分类器偏差方面的巨大威力。而**单棵决策树**虽然具有极高的可解释性，但在处理复杂曲线边界时，受限于其直角切分的本质，性能略逊一筹。
